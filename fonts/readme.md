# info/credit

These fonts were made by the nerd font project here:
https://github.com/ryanoasis/nerd-fonts

I'm including the files straight in my configs repo because I always forget which version of the file I've used on other machines so this is to enforce consistent use of fonts across computers.

## v3 naming convention

As of v3, Nerd Fonts ships three sub-families per font:
- **FiraCode Nerd Font** — standard, double-width icons (good for editors/GUIs)
- **FiraCode Nerd Font Mono** — single-width icons (use this in terminals — correct grid alignment)
- **FiraCode Nerd Font Propo** — proportional spacing (not for coding)

Terminal configs (WezTerm, Tabby, VSCode integrated terminal) reference `FiraCode Nerd Font Mono`.
